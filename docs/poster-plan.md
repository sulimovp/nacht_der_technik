## Title block

- **Titel:** `HACK THE QUANTUM`
- **Untertitel:** `Wie aus Licht ein Bit wird`

## "So geht's" — the 3 steps (very short, with icons)

1. **Mach Licht.** Halte deine Taschenlampe oder deine Hand über den Sensor.
2. **Versuche zu gewinnen.** Erzeuge eine **0** oder eine **1** — du entscheidest das Ziel.
3. **Drei Runden.** Von Runde zu Runde wird es schwerer, das Ergebnis zu steuern.

## Hero graphic — "Wer hat die Kontrolle?"

Three colour blocks showing how control fades:

| 🟢 **Runde 1** | 🟡 **Runde 2** | 🔵 **Runde 3** |
|---|---|---|
| **Hell oder dunkel** | **Versteckte Regel** | **Zufall** |
| Hell = 1, dunkel = 0. Ganz einfach — du gewinnst fast immer. | Nur ein winziges Detail zählt. Plötzlich klappt es nicht mehr so gut. | Die Maschine mischt winzige Schwankungen. Niemand kann das Ergebnis vorhersagen. |
| *du steuerst* 💪 | *halb-halb* 🤔 | *reiner Zufall* 🎲 |

## "Warum ist das spannend?" (3 lines, public-friendly)

- Aus Physik (Licht) entsteht digitale Information (0 und 1).
- Manche Dinge kann man steuern — manche sind **grundsätzlich nicht vorhersagbar**.
- Genau dieser Zufall ist wichtig für **künstliche Intelligenz, Sicherheit im Internet und
  Quantentechnologie** — die Technik der Zukunft. *(Motto: Create Future!)*
